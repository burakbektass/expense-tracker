## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/budgetflow.git
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)
